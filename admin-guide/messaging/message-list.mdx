# Message List

**In a nutshell:** View all your messages in one place, track delivery status, manage drafts, and handle replies from a centralized message management interface.

## What is the Message List?

The message list is your central hub for managing all communications. It shows every message you've sent or received, along with delivery status, read receipts, and reply tracking. Think of it as your email inbox and sent folder combined.

## Key Features

### **Message Overview**

- **All messages** in one organized list
- **Delivery status** for each message
- **Read receipts** showing who opened messages
- **Reply tracking** for conversation threads

### **Message Actions**

- **View message details** and full content
- **Resend failed messages** automatically
- **Delete draft messages** you no longer need
- **Reply to messages** to continue conversations

### **Status Tracking**

- **Real-time updates** on message delivery
- **Read status** for each recipient
- **Reply notifications** when responses arrive
- **Failed message alerts** for troubleshooting

## Message Statuses

### **Draft Messages**

- **Saved but not sent** messages
- **Can be edited** and sent later
- **Appear in your list** for easy access
- **Perfect for complex messages** you're working on

### **Sent Messages**

- **Successfully delivered** to all recipients
- **Show delivery confirmation** status
- **Track read receipts** from recipients
- **Cannot be edited** once sent

### **Delivered Messages**

- **Confirmed delivery** to recipient's inbox
- **Higher engagement** than just "sent"
- **Shows message reached** the destination
- **Next step is** waiting for opens

### **Opened Messages**

- **Recipients have read** the message
- **Shows engagement** with your content
- **Track open rates** for campaign analysis
- **Indicates successful** message delivery

### **Failed Messages**

- **Delivery failed** for some reason
- **Can be resent** to try again
- **Check recipient details** for issues
- **Common causes** include invalid emails

### **Replied Messages**

- **Recipients have responded** to your message
- **Shows active engagement** with your content
- **Continue conversation** by replying back
- **Track response rates** for analysis

## Viewing Messages

### **Message List View**

- **All messages** displayed in a table
- **Sort by date** to see recent messages first
- **Filter by status** to find specific types
- **Search by subject** or recipient name

### **Message Details**

- **Click any message** to view full details
- **See complete content** and formatting
- **View recipient list** and delivery status
- **Check reply thread** if applicable

### **Recipient Information**

- **See who received** each message
- **Track individual** read status
- **View delivery details** for each recipient
- **Identify failed** deliveries quickly

## Message Actions

### **View Message**

1. **Click on any message** in the list
2. **See full content** and details
3. **View recipient information** and status
4. **Check reply thread** if available

### **Resend Failed Messages**

1. **Find failed messages** in the list
2. **Click "Resend"** on the failed message
3. **Confirm resend** in the dialog
4. **Message attempts** delivery again

### **Delete Messages**

1. **Select draft or failed** messages
2. **Click "Delete"** to remove
3. **Confirm deletion** in the dialog
4. **Message is permanently** removed

### **Reply to Messages**

1. **Click "Reply"** on received messages
2. **Original sender** becomes recipient
3. **Subject line** prefixed with "Re:"
4. **Compose your response** and send

## Troubleshooting

### **Common Issues**

- **Messages not sending** - Check recipient emails
- **High failure rates** - Verify email addresses
- **Low open rates** - Improve subject lines
- **No replies** - Check message content

### **Failed Message Causes**

- **Invalid email addresses** - Check recipient details
- **Spam filters** - Avoid spam trigger words
- **Server issues** - Try resending later
- **Recipient inbox full** - Contact recipient directly

### **Delivery Problems**

- **Check internet connection** for sending issues
- **Verify recipient emails** are correct
- **Try resending** failed messages
- **Contact support** for persistent issues

## Getting Started

1. **View your message list** to see all communications
2. **Check message statuses** to understand delivery
3. **Resend any failed** messages that need attention
4. **Reply to messages** that require responses
5. **Clean up drafts** and organize your messages

The message list is your command center for all communications. Use it to track delivery, manage conversations, and ensure your messages reach their intended recipients successfully.
