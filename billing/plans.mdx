In a nutshell, the membership plans system lets you create and manage different pricing tiers for your club members, from one-time passes to recurring monthly subscriptions with flexible billing options.

## Overview

The membership plans system is the foundation of your club's revenue model. You can create various types of memberships, set pricing structures, define usage limits, and track which members are subscribed to which plans.

Key capabilities:

- **Plan Creation**: Set up different membership types and pricing
- **Billing Frequencies**: One-time, weekly, monthly, quarterly, or annual billing
- **Usage Limits**: Set maximum uses for passes and memberships
- **Sports Integration**: Associate plans with specific sports or activities
- **Revenue Tracking**: Monitor plan performance and revenue
- **Member Management**: See which members are subscribed to each plan

### Understanding the Plans Grid

The plans grid displays:

- **Name**: Plan title and description
- **Frequency**: Billing frequency (Monthly, Annually, One-time)
- **Type**: Plan category or type
- **Price**: Cost per billing period
- **Signup Fee**: One-time fee when joining
- **Sessions**: Usage limits (if applicable)
- **Validity**: How long the plan is valid
- **Sports**: Associated sports or activities
- **Status**: Active or inactive
- **Actions**: Edit or delete options

### Plan Statistics

The top of the page shows:

- **Total Plans**: Number of plans created
- **Active Plans**: Currently active plans
- **Average Price**: Mean price across all plans
- **Recurring Plans**: Plans with recurring billing

## Creating Plans

### Adding New Plans

1. Click **"Add Plan"** in the top right
2. Fill in the plan details:
   - **Plan Name**: Descriptive title
   - **Description**: Detailed plan information
   - **Club**: Select the club location
   - **Price**: Cost per billing period
   - **Signup Fee**: Optional one-time fee
   - **Billing Frequency**: How often to charge
   - **Type**: Plan category (General, Class, Court, etc.)
   - **Max Uses**: Usage limits (optional)
   - **Validity Period**: Days the plan is valid
   - **Sports**: Associated activities
   - **Notes**: Additional information
3. Click **"Save"** to create the plan

### Plan Types

#### Recurring Plans

- **Monthly**: Charged every month
- **Quarterly**: Charged every 3 months
- **Annually**: Charged once per year
- **Weekly**: Charged every week
- Perfect for ongoing memberships

#### One-Time Plans

- **Single Payment**: Paid once upfront
- **Usage-Based**: Limited number of uses
- **Time-Limited**: Valid for specific period
- Great for passes and packages

### Billing Frequencies

- **One-time**: Single payment, no renewal
- **Weekly**: Charged every 7 days
- **Monthly**: Charged every 30 days
- **Quarterly**: Charged every 90 days
- **Annually**: Charged every 365 days

## Plan Configuration

### Usage Limits

#### Limited Plans

## Common Tasks

### Setting Up Basic Memberships

1. Create a **Monthly** plan for regular members
2. Set appropriate pricing for your market
3. Choose **Unlimited** usage for full access
4. Associate with all relevant sports
5. Set as **Active** to make it available

### Creating Class Passes

1. Create a **One-time** plan
2. Set the number of classes included
3. Choose **Class** as the type
4. Set validity period (e.g., 30 days)
5. Associate with specific sports

### Setting Up Court Time

1. Create **One-time** or **Monthly** plans
2. Set usage limits based on hours
3. Choose **Court** as the type
4. Set appropriate pricing per hour
5. Associate with court sports

### Creating Premium Tiers

1. Create multiple plans with different features
2. Use higher pricing for premium plans
3. Include additional benefits or services
4. Set different usage limits
5. Market to different member segments

## Tips & Best Practices

- **Clear Naming**: Use descriptive names that members understand
- **Pricing Strategy**: Research local market rates for competitive pricing
- **Usage Limits**: Set realistic limits based on facility capacity
- **Validity Periods**: Don't make plans too restrictive
- **Sports Association**: Only include relevant sports for each plan
- **Regular Review**: Monitor plan performance and adjust as needed

## Troubleshooting

**Issue**: Cannot create plan
**Solution**: Ensure all required fields are filled and the club is selected

**Issue**: Plan not showing in member signup
**Solution**: Check that the plan is set to **Active** status

**Issue**: Usage limits not working
**Solution**: Verify that **Max Uses** is set and members are using the correct plan

**Issue**: Billing frequency not updating
**Solution**: Check that the plan doesn't have active memberships before changing frequency

**Issue**: Revenue not tracking correctly
**Solution**: Verify that the **Revenue Account** is properly assigned

**Issue**: Plan not appearing in reports
**Solution**: Ensure the plan is active and has been used by members
