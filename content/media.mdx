**In a nutshell:** The Media library stores images and files you use in documents, website pages, and emails so you can reuse them and keep branding consistent.

## Overview

Use **Content → Media** to upload, organize, and manage media files. Files are typically stored in your configured storage (e.g. Cloudinary). You can browse by type, search, and insert media into documents and website content.

## Key features

- **Upload** — Add images and other files; supported formats depend on your integration.
- **Library** — Browse and search media; view thumbnails and metadata.
- **Usage** — Insert media into documents, website blocks, and email templates.
- **Organization** — Optional folders or tags to keep media findable.
