**In a nutshell:** The Website section lets you design and update your club's public website—pages, layout, and content—so members and visitors see an up-to-date, branded site.

## Overview

Use **Content → Website** to open the website builder. You can edit the structure and content of your club site: add or reorder pages, change blocks (text, images, classes, signup), and preview before publishing. The site is typically available at your member portal or custom domain.

## Key features

- **Page structure** — Add, remove, and reorder pages and sections.
- **Blocks** — Use content blocks (text, images, class schedule, signup forms, etc.) to build each page.
- **Preview** — See how the site looks before publishing.
- **Publish** — Push changes live so the public site updates.
