**In a nutshell:** Campaigns let you create and run marketing campaigns (e.g. email or messaging) with AI-assisted targeting and content so you can reach the right members at the right time.

## Overview

Use **Marketing → Campaigns** to create new campaigns, set audience and schedule, and use AI tools to refine messaging. You can base campaigns on segments, tags, or membership status and track performance from the campaign list.

## Key features

- **Campaign list** — View all campaigns, status (draft, scheduled, sent), and basic metrics.
- **Create campaign** — Define audience, channel, content, and schedule.
- **AI assistance** — Use AI to suggest or improve message content and targeting.
- **Reporting** — See opens, clicks, and engagement per campaign.

## Campaign types

### Segment-based

Target a defined audience segment with a one-time or recurring message. Use this for newsletters, promotions, or re-engagement campaigns aimed at a specific group of members.

### Review request

Automatically send review request messages to members after a class or area booking. Drives Google reviews and reputation management without manual effort.

### Reminder

Automatically send booking reminders to members before their upcoming class or session.

Two default reminder campaigns are created automatically when you set up your organization:
- **24 hours before** — delivered via email, push notification, and in-app
- **1 hour before** — delivered via push notification and in-app

You can create additional reminders or adjust the defaults. When creating or editing a Reminder campaign, configure:

- **Time before booking** — Minutes before the booking the reminder is sent. Minimum 15 minutes, maximum 10,080 minutes (7 days). Common values: `60` (1 hour), `1440` (24 hours), `2880` (48 hours).
- **Notification channels** — Choose one or more delivery channels:
  - **In-app** — notification bell inside the member portal
  - **Push** — mobile and browser push notifications
  - **Email** — email to the member
  - **SMS** — text message (requires SMS to be configured)
- **Clubs** — Optionally restrict the reminder to specific club locations. Leave empty to apply to all clubs.

Reminder campaigns run automatically in the background. Each member receives the reminder once per booking.
