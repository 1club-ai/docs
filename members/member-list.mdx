The Member List provides a comprehensive view of all your club members with powerful search, filtering, and management capabilities.

## Accessing the Member List

1. Navigate to **Members** in the main menu
2. Click on **Member List** (or it may be the default view)
3. View all members in a sortable, searchable table

## Member List Features

### Viewing Members

- **Complete Member Information** - Name, email, phone, membership status
- **Quick Status Indicators** - Visual indicators for membership status
- **Last Activity** - When the member last visited or interacted
- **Membership Details** - Current plan, renewal date, payment status

### Search and Filter

- **Quick Search** - Search by name, email, or phone number
- **Advanced Filters** - Filter by membership status, plan type, activity level
- **Date Range Filters** - Filter by join date, last visit, renewal date
- **Custom Field Filters** - Filter by any custom field values

### Sorting Options

- **Name** - Alphabetical sorting by first or last name
- **Join Date** - Sort by when members joined
- **Last Activity** - Sort by most recent activity
- **Membership Status** - Group by active, expired, pending
- **Payment Status** - Sort by payment status

## Member Actions

### Individual Member Actions

- **View Profile** - Click on member name to view full profile
- **Edit Information** - Update member details
- **Send Message** - Send email directly to member
- **View History** - See complete member activity
- **Manage Membership** - Change plans, pause, or cancel

### Bulk Actions

- **Select Multiple Members** - Use checkboxes to select multiple members
- **Bulk Email** - Send messages to selected members
- **Bulk Status Changes** - Change membership status for multiple members
- **Export Data** - Export selected member information
- **Bulk Operations** - Perform actions on multiple members

## Member Status Indicators

### Active Members

- **Green Indicator** - Currently active membership
- **Check Mark** - Up-to-date with payments
- **Recent Activity** - Visited within the last 30 days

### Expired Members

- **Red Indicator** - Membership has expired
- **Warning Icon** - Payment overdue or membership lapsed
- **No Recent Activity** - Haven't visited recently

### Pending Members

- **Yellow Indicator** - Membership pending activation
- **Clock Icon** - Waiting for payment or approval
- **New Member** - Recently signed up

### Suspended Members

- **Orange Indicator** - Membership temporarily suspended
- **Pause Icon** - Membership on hold
- **Special Status** - Requires attention

## Member Information Display

### Basic Information

- **Full Name** - Member's complete name
- **Email Address** - Primary contact email
- **Phone Number** - Contact phone number
- **Join Date** - When member joined the club

### Membership Information

- **Current Plan** - Active membership plan name
- **Plan Type** - Monthly, annual, or other
- **Renewal Date** - When membership expires
- **Payment Status** - Current payment status

### Activity Information

- **Last Visit** - Most recent check-in date
- **Visit Count** - Total number of visits
- **Engagement Score** - Member activity level
- **Preferred Areas** - Most used facilities

## Troubleshooting

### Member Not Appearing

1. Check filter settings
2. Verify member status
3. Confirm search criteria
4. Check user permissions
5. Contact support if needed

### Slow Loading

1. Reduce number of displayed columns
2. Use filters to limit results
3. Check internet connection
4. Clear browser cache
5. Contact technical support

### Export Issues

1. Verify export permissions
2. Check data selection
3. Confirm file format
4. Try smaller data sets
5. Contact support for assistance

The Member List is your central hub for member management, providing all the tools needed to effectively manage your club's membership base.
