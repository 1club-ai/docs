Manage membership plans, track active memberships, and handle renewals and changes.

## Membership Plans

### Creating Plans

1. Navigate to **Billing** → **Plans**
2. Click **Add Plan**
3. Configure plan details:
   - Plan name and description
   - Monthly price and billing frequency
   - Signup fees (optional)
   - Active status
4. Save the plan

### Plan Types

- **Unlimited Plans** - Access to all facilities
- **Limited Plans** - Restricted access or usage
- **Class-specific Plans** - Access to specific classes
- **Area-specific Plans** - Access to specific areas

## Managing Memberships

### Active Memberships

- View all active memberships
- Track renewal dates
- Handle membership changes
- Process cancellations

### Membership Status

- **Active** - Currently active membership
- **Expired** - Membership has expired
- **Suspended** - Temporarily suspended
- **Cancelled** - Membership cancelled

<Note>
  An exhausted or no-longer-active membership still honors bookings that were created against it while it was active - the booking debited a use at booking time, so check-in for that specific booking goes through even if the membership has since hit zero uses or flipped out of active status. See [Attendance](/members/attendance) for details.
</Note>

## Best Practices

### Plan Design

- Create clear, simple plans
- Offer different price points
- Include signup incentives
- Set appropriate usage limits

### Member Communication

- Send renewal reminders
- Notify of plan changes
- Provide plan benefits information
- Handle upgrade/downgrade requests
