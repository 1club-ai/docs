**In a nutshell:** Create reusable email and SMS templates to save time and ensure consistent messaging across your communications.

## What are Message Templates?

Message templates are pre-written messages that you can use as starting points for your communications. They help you maintain consistency, save time, and ensure important information is always included in your messages.

## Key Features

### **Template Management**

- **Create custom templates** for your specific needs
- **Edit existing templates** to keep them current
- **Copy templates** to create variations
- **Delete unused templates** to keep your list clean

### **Template Categories**

- **Welcome messages** for new members
- **Class reminders** for upcoming sessions
- **Promotional content** for special offers
- **General announcements** for club updates

### **Content Types**

- **HTML templates** with rich formatting
- **Plain text templates** for simple messages
- **SMS templates** for mobile communications
- **Email templates** for detailed content

## Creating Templates

### **Basic Template Setup**

1. Go to **Messaging** → **Templates**
2. Click **Add Template**
3. **Enter template name** and description
4. **Choose content type** (HTML or text)
5. **Write your template content**
6. **Save the template**

### **Template Content**

- **Subject line** for email templates
- **Message body** with your content
- **Placeholder variables** for personalization
- **Rich formatting** for HTML templates

### **Template Variables**

Variables are written as `{{variableName}}` and substituted at send time. Use the **Insert variable** picker in the editor toolbar to browse and insert variables without typing them by hand. The picker groups variables by category (Contact, Organization, Booking, Membership) and shows a short description for each.

For automations, the available variables depend on the **trigger**. Booking-triggered messages get the booking variables; membership-triggered messages get the membership variables. Variables that aren't available for the current trigger ship as literal `{{variableName}}` text, so always insert from the picker rather than typing.

#### Available everywhere

Available on every trigger and in any manual message.

| Variable | Description |
| --- | --- |
| `{{contactFirstName}}` | Recipient's first name (use for greetings) |
| `{{contactName}}` | Recipient's full name |
| `{{contactLastName}}` | Recipient's last name |
| `{{contactEmail}}` | Recipient's email address |
| `{{organizationName}}` | Your organization's name |
| `{{organizationEmail}}` | Your organization's email |
| `{{organizationPhone}}` | Your organization's phone number |
| `{{clubName}}` | Club name |
| `{{orgBaseUrl}}` | Your organization's base URL |
| `{{orgLogoUrl}}` | Your organization's logo URL |

#### Booking triggers

Available on booking-related triggers: booking created, cancelled, no-show, checked-in, and upcoming reminders.

| Variable | Description |
| --- | --- |
| `{{entityName}}` | The booked class, area or instructor (no club name) |
| `{{bookingResource}}` | Joined display of class, area, instructor and club on the booking |
| `{{bookingDate}}` | Booking date, e.g. "Mar 12, 2026" |
| `{{bookingTime}}` | Booking time range, e.g. "10:00 - 11:00" |
| `{{bookingStartTime}}` | Booking start time |
| `{{bookingEndTime}}` | Booking end time |
| `{{bookingDuration}}` | Booking length, e.g. "1h" or "1h 30min" |
| `{{accessPin}}` | The member's access PIN, or empty if they don't have one |

#### Membership triggers

Available on membership-related triggers: created, cancelled, and expiring.

| Variable | Description |
| --- | --- |
| `{{planName}}` | Membership plan name |
| `{{renewalPrice}}` | Renewal amount with currency |
| `{{membershipExpiresAt}}` | Membership expiration date |
| `{{daysUntilExpiration}}` | Days until membership expires |

#### Membership expiring only

| Variable | Description |
| --- | --- |
| `{{renewUrl}}` | Deep link to the member portal renewal page |

<Note>
  `{{accessPin}}` returns only the raw PIN (or empty). Write any surrounding label as plain text, for example: `Door code: {{accessPin}}`. The older `{{accessPinLine}}` token that rendered a full English sentence has been removed.
</Note>

## Using Templates

### **In Message Composition**

1. **Start composing** a new message
2. **Select a template** from the dropdown
3. **Template content** automatically fills in
4. **Customize the content** as needed
5. **Send or schedule** your message

### **Template Customization**

- **Edit subject lines** for specific campaigns
- **Modify content** to match your needs
- **Add personal touches** to make it unique
- **Keep the template structure** for consistency

## Template Management

### **Editing Templates**

1. **Click Edit** on any template
2. **Modify the content** as needed
3. **Update the name** or description
4. **Save changes** to update the template

### **Copying Templates**

1. **Click Copy** on any template
2. **Template opens** with "(Copy)" in the name
3. **Modify the content** for your new template
4. **Save as a new template**

### **Deleting Templates**

1. **Click Delete** on any template
2. **Confirm deletion** in the dialog
3. **Template is permanently removed**
4. **Cannot be undone**

## Template Best Practices

### **Content Guidelines**

- **Keep templates concise** and to the point
- **Use clear, friendly language** that matches your brand
- **Include all necessary information** in the template
- **Test templates** before using them widely

### **Organization Tips**

- **Use descriptive names** for easy identification
- **Group similar templates** by category
- **Update templates regularly** to keep them current
- **Remove unused templates** to avoid confusion

### **Personalization**

- **Use template variables** for personalization
- **Test with real data** to ensure variables work
- **Keep personalization natural** and not forced
- **Balance personalization** with template efficiency

## Template Examples

### **Welcome Email Template**

```text
Subject: Welcome to {{organizationName}}!

Hi {{contactFirstName}},

Welcome to {{organizationName}}! We're excited to have you as a new member.

If you have any questions, don't hesitate to reach out at {{organizationEmail}}.

Best regards,
The {{organizationName}} Team
```

### **Booking Reminder Template**

Use on a booking trigger so that `{{entityName}}`, `{{bookingDate}}`, `{{bookingTime}}` and `{{bookingDuration}}` are available.

```text
Subject: Reminder: {{entityName}} at {{clubName}}

Hi {{contactFirstName}},

This is a friendly reminder for your upcoming session:

- What: {{entityName}}
- When: {{bookingDate}}, {{bookingTime}} ({{bookingDuration}})
- Where: {{clubName}}
- Door code: {{accessPin}}

Please arrive a few minutes early to check in.

See you there!
The {{organizationName}} Team
```
